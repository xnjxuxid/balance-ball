# 平衡球控制系统

基于 STM32F103C8 的串级 PID 平衡球控制系统。通过激光测距传感器检测小球位置，驱动直流电机调节轨道倾斜角度，使小球稳定在目标位置。

## 硬件清单

| 模块 | 型号/规格 | 用途 |
|------|-----------|------|
| 主控 | STM32F103C8T6 最小系统板 | 控制核心 |
| 激光测距 | VL53L1X | 检测小球位置（mm） |
| 直流电机 + 编码器 | 带编码器的直流电机 | 驱动轨道倾斜 |
| 电机驱动 | L298N / TB6612 / DRV8833 等 | PWM 驱动电机 |
| OLED 显示 | 0.96寸 SSD1306 (128×64, I2C) | 显示运行状态 |
| 按键 | 2 个 | 调节目标距离 |
| IMU | MPU6050（预留，暂未使用） | 可用于角度融合 |

## 接线图

### 核心接线

| STM32 引脚 | 连接模块 | 功能 |
|------------|----------|------|
| **PA0** | 按键 B2 | 按键输入（目标距离 +10mm） |
| **PA2** | 电机驱动 PWM 输入 | TIM2_CH3 PWM 输出（10KHz） |
| **PA3** | LED1 | 状态指示灯 |
| **PA4** | 电机驱动 IN1 | 方向控制 |
| **PA5** | 电机驱动 IN2 | 方向控制 |
| **PA6** | 编码器 A 相 | TIM3_CH1 编码器模式 |
| **PA7** | 编码器 B 相 | TIM3_CH2 编码器模式 |
| **PA8** | LED2 | 状态指示灯 |
| **PA9** | USB转TTL TX | 串口发送（115200） |
| **PA10** | USB转TTL RX | 串口接收（115200） |
| **PB0** | 按键 B1 | 按键输入（目标距离 -10mm） |
| **PB8** | OLED SCL | 软件 I2C 时钟 |
| **PB9** | OLED SDA | 软件 I2C 数据 |
| **PB10** | MPU6050 SCL | 软件 I2C 时钟 |
| **PB11** | MPU6050 SDA | 软件 I2C 数据 |

### 电机驱动接线

```
STM32 PA2  → 驱动模块 PWM/ENA（速度控制）
STM32 PA4  → 驱动模块 IN1    （方向控制）
STM32 PA5  → 驱动模块 IN2    （方向控制）
驱动模块 OUT1/OUT2 → 直流电机
驱动模块 VM → 电机电源（7~12V，不要用 STM32 的 3.3V）
驱动模块 GND → STM32 GND（必须共地）
```

### VL53L1X 接线

```
VL53L1X SDA → STM32 PB12（VL53L1X_I2C.c 中定义）
VL53L1X SCL → STM32 PB13
VL53L1X VCC → 3.3V
VL53L1X GND → GND
```

> **注意**：VL53L1X 使用独立的软件 I2C（PB12/PB13），与 MPU6050（PB10/PB11）和 OLED（PB8/PB9）互不冲突。

## 软件架构

### 串级 PID 控制框图

```
目标距离 (180mm)
    │
    ▼
┌──────────┐
│ 外环 PID │ ← VL53L1X 激光测距反馈
│ 距离→角度 │
└────┬─────┘
     │ 目标角度 (TargetSmooth)
     ▼
┌──────────┐
│ 内环 PID │ ← 编码器位置反馈
│ 角度→PWM │
└────┬─────┘
     │ PWM 输出 (Out, -100~+100)
     ▼
  直流电机
```

- **外环**（主循环，~50Hz）：激光测距 → 距离误差 → 计算目标角度
- **内环**（TIM4 中断，~33Hz）：编码器 → 角度误差 → PWM 输出驱动电机

### 控制特性

- **低通滤波**：距离、速度、微分项均有一阶低通滤波，消除噪声
- **积分管理**：大误差清零、误差方向反转清零、正常范围累积，防止积分饱和
- **最小 PWM**：误差较大时强制最低 PWM 克服电机静摩擦
- **死区**：角度误差 < 0.3 脉冲时停止输出，防止到位后抖动
- **激光丢失保护**：连续 30 帧无效数据自动停电机，恢复后自动解除

## 编译与烧录

### 环境要求

- Keil MDK 5.x
- STM32F10x 标准外设库（已包含在 `Library/` 目录）

### 编译步骤

1. 用 Keil 打开 `Project.uvprojx`
2. 点击 **Build**（F7）编译
3. 确认底部输出 `0 Error(s), 0 Warning(s)`

### 烧录

1. ST-Link 连接 STM32（SWD 接口）
2. Keil 中点击 **Download**（F8）
3. 或使用 ST-Link Utility 烧录 `Objects/Project.hex`

### Keil 设置确认

- **Target → Code Generation → Use MicroLIB**：必须勾选（printf 重定向需要）
- **Target → ARM Compiler → Compiler version**：建议使用 V5 编译器

## 串口调试

### 连接方式

USB转TTL 模块：
- TX → PA10（STM32 RX）
- RX → PA9（STM32 TX）
- GND → GND

串口助手设置：**波特率 115200，8N1，发送新行（\r\n）**

推荐工具：VOFA+、SSCOM、XCOM、PuTTY

### 实时数据输出

每 200ms 自动输出一行：

```
Dis:180 Filt:180 Err:0.1 Vel:-0.1 Tar:-5.5 Act:-6.0 Out:38.0
```

| 字段 | 含义 | 单位 |
|------|------|------|
| Dis | 激光测距原始值 | mm |
| Filt | 低通滤波后距离 | mm |
| Err | 距离误差（目标 - 实际） | mm |
| Vel | 距离变化速度（滤波后） | mm/周期 |
| Tar | 外环输出的目标角度 | 编码器脉冲 |
| Act | 电机实际角度 | 编码器脉冲 |
| Out | PWM 输出值 | -100 ~ +100 |

### 串口调参命令

发送命令后会回显确认，如发送 `KP=5.0`，回复 `AngKp=5.000`。

#### 内环参数（角度 PID）

| 命令 | 变量 | 默认值 | 含义 | 调节方法 |
|------|------|--------|------|----------|
| `KP=` | AngKp | 3.5 | 比例系数 | 电机跟不上→加大；电机抖动→减小 |
| `KI=` | AngKi | 0.028 | 积分系数 | 到位有固定偏差→加大；过冲→减小 |
| `KD=` | AngKd | 1.2 | 微分系数 | 配合 Kp 抑制振荡→加大 |
| `BZ=` | AngDeadzone | 0.3 | 死区（脉冲） | 到位后微动→加大（0.5~1.0） |
| `BI=` | AngBias | 18 | 摩擦补偿 | 低速不动→加大；起步过猛→减小 |
| `MP=` | AngMinPWM | 40 | 最小 PWM | 电机推不动→加大；到位抖动→减小 |

#### 外环参数（距离 PID）

| 命令 | 变量 | 默认值 | 含义 | 调节方法 |
|------|------|--------|------|----------|
| `DKP=` | DistKp | 0.35 | 比例系数 | 小球反应慢→加大；来回跑→减小 |
| `DKI=` | DistKi | 0.018 | 积分系数 | 停不准有偏差→加大；慢速振荡→减小 |
| `DKD=` | DistKd | 0.40 | 微分系数 | 冲过头→加大；太肉→减小 |
| `TD=` | DistTarget | 180 | 目标距离（mm） | 也可用按键 ±10mm 调节 |
| `AM=` | AngleMaxP | 80 | 最大正角度 | 限制轨道倾斜范围 |
| `AN=` | AngleMaxN | -80 | 最大负角度 | 限制轨道倾斜范围 |

### 命令示例

```
KP=5.0        ← 把角度环 Kp 改为 5.0
DKP=0.5       ← 把距离环 Kp 改为 0.5
TD=200        ← 把目标距离改为 200mm
MP=50         ← 把最小 PWM 改为 50
```

> **注意**：串口修改的参数掉电或烧录新代码后会恢复为代码中的默认值。

|------|------|------|
| B1 | PB1 | 目标距离 -10mm（最低 50mm） |

## 调参指南

### 调参顺序

**先内环，后外环。** 内环不稳，外环调得再好也没用。

### 第一步：让电机能动

```
MP=50         ← 如果电机推不动，加大最小 PWM
BI=25         ← 如果还不动，加大摩擦补偿
```

### 第二步：调内环（让电机跟得上目标）

```
KP=5.0        ← 电机反应慢就加大
KD=2.0        ← 电机抖动就加大微分
KI=0.05       ← 到位有偏差就加大积分
```

### 第三步：调外环（让小球停在目标位置）

```
DKP=0.5       ← 小球追目标太慢就加大
DKD=0.6       ← 小球冲过头就加大
DKI=0.03      ← 小球停不准有偏差就加大
```

### 调参原则

1. **一次只改一个参数**，观察几秒再决定下一步
2. **每次改动不超过原来的 50%**
3. 先让系统能工作，再逐步优化
4. 如果完全不知道怎么调，先恢复默认值：`KP=3.5 KI=0.028 KD=1.2 DKP=0.35 DKI=0.018 DKD=0.40`

## 常见问题

### Q: 电机完全不动

- 检查驱动模块供电（VM 接了没？）
- 检查 PA2 是否接到驱动模块 PWM 输入
- 串口发 `MP=60` 加大最小 PWM
- 串口发 `BI=30` 加大摩擦补偿

### Q: 电机嗡嗡响但不转

- 最小 PWM 不够克服静摩擦，发 `MP=50` 或更大
- 如果 Out 在 0 和 MinPWM 之间反复跳，说明到了临界值，再加一点

### Q: 小球来回跑停不下来

- 外环太灵敏，减小 DKP：`DKP=0.2`
- 或加大微分阻尼：`DKD=0.8`

### Q: 小球到不了目标位置，停在半路

- 外环太弱，加大 DKP：`DKP=0.5`
- 或加大积分：`DKI=0.03`

### Q: 电机一直往一个方向跑到头

- 检查编码器接线是否正确（A/B 相接反会导致方向错误）
- 串口看 Act 值是否在变化，不变说明编码器没信号

### Q: 串口没有输出

- 检查 TX/RX 是否接反
- 波特率是否为 115200
- 是否勾选了 Keil 的 Use MicroLIB

### Q: 串口发命令没反应

- 确保发送了换行符（\r\n 或 \n）
- 检查 RX 接线（USB转TTL TX → STM32 PA10）
